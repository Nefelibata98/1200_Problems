#### 0002 反转单链表

OJ 链接：https://leetcode-cn.com/problems/reverse-linkedlist/description/  

**题目描述：**

反转一个单链表。

**示例:**

```
输入: 1->2->3->4->5->NULL
输出: 5->4->3->2->1->NULL
```

模板：

```java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */
```
